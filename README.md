# Indigenous Culture Atlas (原住民文化地圖集)

這是一個專注於**原住民文化知識工程**的開源專案。透過結合 SQLite、GIS 與知識圖譜，我們致力於建構一個具備空間維度與邏輯關聯的數位民族誌底層。

## 🚀 專案核心
- **Structured Database**: 嚴謹的關聯式資料庫設計，支援多族群、多語彙與證據溯源。
- **Spatial Intelligence**: 整合傳統領域圖資與流域分析。
- **Knowledge Graph**: 紀錄神話、圖騰與社會結構間的複雜語意連結。
- **Methodology Book**: 隨附《數位民族誌方法論》專書，記錄研究與建模的核心思維。

## 📂 目錄結構
- `TOC.md`: 《數位民族誌方法論》專書大綱與目錄。
- `/db`: 資料庫 Schema 與初始化腳本。
- `/docs`: 設計規範、SOP 與 AI 協作指引。
- `/scripts`: 自動化匯入、轉換與分析工具。
- `/data`: 已整理的開放資料集 (Open Data)。

## 📖 快速開始
1. 閱讀 [TOC.md](TOC.md) 了解方法論。
2. 檢視 [init_schema.sql](db/init_schema.sql) 了解資料結構。
3. 參考 [AI Agent Guide](docs/ai_agent_guide.md) 進行協作開發。
